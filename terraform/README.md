# Развёртывание инфраструктуры Kubernetes через Terraform

Для запуска проекта необходимо создать **две ноды** в разделе **«Облачные серверы»** в Selectel.

Для развёртывания используется **Terraform**: создаются **master** и **worker** ноды.

Обе ноды будут:

- находиться в **одной локальной сети**
- иметь **внешние IP-адреса**

---

# 1. Заполнение переменных

Заполните файл:

```
terraform.tfvars
```

вашими **учётными данными из Selectel**.

В файле `vars.tf` можно задать параметры будущих серверов.

Сейчас указаны следующие параметры:

- **CPU:** 2  
- **RAM:** 4 GB  
- **Disk:** 8 GB  

Также в `vars.tf` задаётся **количество серверов** — в нашем случае:

```
2
```

---

# 2. Запуск Terraform

После заполнения учётных данных выполните в директории `terraform` следующие команды:

```bash
terraform init
```

```bash
terraform validate
```

```bash
terraform plan
```

Должно показать создание примерно:

```
~15 ресурсов
```

После этого выполните:

```bash
terraform apply
```

Подтвердите выполнение:

```
yes
```

---

# 3. Проверка результата

После успешного выполнения в облаке **Selectel** появится проект:

```
k8s_server
```

В котором будут созданы **два сервера**.

---

# 4. Назначение внешнего IP

В панели управления **Selectel** выберите любой сервер, который будет **master-нодой**, и назначьте ему **внешний IP-адрес**.
