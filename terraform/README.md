Для запуска проекта необходимо создать две ноды в разделе «Облачные серверы» в Selectel.
Для развёртывания мы будем использовать Terraform: создадим master и worker ноды.

Обе ноды будут находиться в одной локальной сети и иметь внешние IP-адреса.

1. Заполнение переменных
Заполните файл terraform.tfvars вашими учетными данными из Selectel.

В файле vars.tf можно задать параметры будущих серверов.
Сейчас там указаны следующие параметры: 2 CPU, 4 GB RAM, диск 8 GB

Также в vars.tf задаётся количество серверов — в нашем случае 2.

2. Запуск Terraform
После заполнения учётных данных выполните в директории terraform следующие команды:
terraform init
terraform validate
terraform plan   # должно показать создание ~15 ресурсов
terraform apply  # подтвердите выполнение (yes)

3. Проверка результата
После успешного выполнения в облаке Selectel появится проект k8s_server, в котором будут созданы два сервера.

4. Назначение внешнего IP
В панели управления Selectel выберите любой сервер, который будет master-нодой, и назначьте ему внешний IP-адрес.

5. Следующий шаг
После этого можно переходить в раздел Ansible.
